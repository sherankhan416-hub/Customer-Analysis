# 📊 Data Analysis Project

## 🔍 Overview

This project demonstrates an end-to-end data analysis workflow, starting from data loading and exploration to visualization and reporting. The goal is to extract meaningful insights from the dataset using Python, SQL, and Power BI, and present the findings in a clear and professional format.

---

## 📁 Dataset

* The dataset is loaded and processed using **Python (Jupyter Notebook)**.
* It contains structured data suitable for analysis (e.g., sales, customer, or operational data).
* Initial inspection includes checking data types, missing values, and overall structure.

---

## 🛠️ Tools & Technologies

* **Python (Jupyter Notebook)** – Data loading, cleaning, and EDA
* **Libraries** – Pandas, NumPy, Matplotlib, Seaborn
* **MySQL Workbench** – SQL queries and data manipulation
* **Power BI** – Interactive dashboard creation
* **Gamma** – Presentation (PPT) creation
* **MS Excel / CSV** – Data storage (if applicable)

---

## ⚙️ Project Steps

### 1. Data Loading

* Imported dataset into Jupyter Notebook using Pandas
* Verified data structure, column names, and data types

### 2. Exploratory Data Analysis (EDA)

* Analyzed distributions and relationships between variables
* Used visualizations (bar charts, histograms, heatmaps)
* Identified trends, patterns, and anomalies

### 3. Data Cleaning

* Handled missing values
* Removed duplicates and irrelevant columns
* Corrected data types and formatting issues

### 4. SQL Analysis (MySQL Workbench)

* Imported cleaned data into MySQL
* Wrote SQL queries to:

  * Aggregate data (SUM, AVG, COUNT)
  * Filter records (WHERE conditions)
  * Join tables (if applicable)
  * Generate business insights

### 5. Dashboard Creation (Power BI)

* Connected Power BI to the dataset
* Built interactive dashboards with:

  * KPIs and summary metrics
  * Filters and slicers
  * Visual charts (bar, line, pie, etc.)

### 6. Reporting & Presentation

* Created a structured report summarizing key findings
* Designed a professional presentation using Gamma
* Highlighted insights, recommendations, and conclusions

## ▶️ How to Run the Project

1. Clone or download the repository
2. Open the Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
3. Install required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
4. Run the notebook step-by-step for EDA and data cleaning
5. Import cleaned data into MySQL Workbench and execute SQL queries
6. Open Power BI file (.pbix) to view the dashboard
7. Review the final report and presentation
